Printing for Staff & Faculty

  • Staff & Faculty printers and photocopiers are named by room location
  • Default printers are assigned based on staff and faculty office location
  • All printers are set to duplex printing by default. To turn duplex printing off, under File, Print menu, click Print Properties, and change settings on Print on Both Sides (Instructions may vary for other applications)

How to Print

Printing from PCs

  1. File, Print
  2. Select Photocopier (e.g. ES-HNES123-copier)
  3. Select Printer Properties or Preferences
  4. Select Detailed Settings tab
  5. Select Job Setup then enter Photocopy/User Code

Printing from Macs

  1. File, Print
  2. Select Photocopier (e.g. ES-HNES123-copier)
  3. Under “copies & pages”, select Job log
  4. Checkmark Enable User Code then enter Photocopy/User Code

Confidential Printing

  1. File, Print
  2. Select Photocopier
  3. Select Printer Properties or Preferences
  4. Select Detailed Settings tab
  5. Select Job Setup, under Job type select locked print
  6. Select Details
  7. Enter User ID and password
  8. Press Printer button on the photocopier, then press Print Jobs
  9. Select your file and press Print
